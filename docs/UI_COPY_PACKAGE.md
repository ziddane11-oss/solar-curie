# 톡캐디 GRAVITY - UI 카피 패키지

## 전체 흐름
**익명 체험 → 재미 충격 → 무료 제한 → 공유 보상 → 로그인 락인 → 위기 순간 유료**

---

## A. 첫 화면 (Drop)
- 헤드라인: **톡을 던져봐**
- 서브1: **카톡 스크린샷 여기로 드래그**
- 서브2(작게): **로그인 없이 바로 가능**
- 보조: **또는 클릭해서 선택**
- 버튼: **데모 체험하기**

## B. 로딩 문구 (랜덤)
- 유사 대화 1,284건 비교 중…
- 상대 반응 패턴 매칭 중…
- 실수 포인트 검사 중…
- 보내기 위험도 계산 중…

## C. 결과 상단
- 확률 아래: **유사 대화 기준 · 현재 멘트 유지 시**
- 판정 아래: **지금은 "밀어붙임"보다 "흐름 유지"가 유리함**

## D. 탭 라벨
- **판정** | **독설** | **액션**

## E. 액션카드 라벨
| 라벨 | 설명 |
|------|------|
| 👍 안전 | 부담 최소 |
| 🧊 밀당 | 여운 남김 |
| 😈 도발 | 반응 유도 |
| ⚠️ 리스크 | 읽씹 가능 |
| 🔥 고위험 | 관계 흔들림 |

## F. 잠금 카드
- 타이틀: **흐름 뒤집는 한 문장**
- 서브: **이 상황에서 제일 흔한 실수 피하게 해줌**
- 버튼: **열어보기**

## G. 무료 제한 모달
- 타이틀: **오늘 무료 분석 다 씀**
- 본문1: **지금은 고위험 멘트라 무료에선 숨김**
- 본문2: **공유하면 +1회 바로 지급함**
- 버튼1: **공유하고 +1 받기**
- 버튼2: **내일 다시 하기**
- 하단: **하루 최대 2회까지**

## H. 공유 보상
- **🎁 공유 보상 +1회 지급됨**
- (작게) **지금 바로 잠금 멘트 열 수 있음**

## I. 로그인 락인 (3번째 사용)
- 타이틀: **내 판정 저장할래?**
- 본문1: **대화 스타일 학습하려면 기록이 필요함**
- 본문2: **로그인하면 히스토리/즐겨찾기 사용 가능**
- 버튼1: **로그인하고 저장**
- 버튼2: **나중에**
