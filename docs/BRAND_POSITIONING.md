# 톡캐디 GRAVITY 브랜드 포지셔닝

## 시장 포지션
```
X축: 재미(가벼움) ↔ 진지(정확함)
Y축: 도구형(문장 생성) ↔ 코치형(판정+행동 설계)
```

### 경쟁 포지션
| 서비스 | 포지션 | 특징 |
|--------|--------|------|
| 일반 AI챗봇 | 진지+도구형 | 문장만 뽑아줌 |
| 연애상담 카페 | 재미+코치형 | 사람이 평가 |
| 멘트 생성기 앱 | 재미+도구형 | 원라이너 |
| **톡캐디** | **재미+코치형** | **판정/독설/액션 3단 구조** |
