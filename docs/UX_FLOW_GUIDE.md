# 톡캐디 GRAVITY - UX 플로우 가이드

## 전체 흐름 (한 줄)
**익명 체험 → 재미 충격 → 무료 제한 → 공유 보상 → 로그인 락인 → 위기 순간 유료**

---

## ① 첫 화면 (Drop Zone)
- 로그인 없이 바로 가능
- 하단 문구: "로그인 없이 바로 가능"

## ② 분석 중 로딩
- "상대 반응 패턴 비교 중…"
- "비슷한 상황 1,284건 분석 중"
- "지금 멘트, 위험 요소 검사 중"

## ③ 결과 화면
- 확률 아래: "유사 대화 기준 · 현재 멘트 유지 시"

## ④ 액션 카드
- 2개 무료 + 1개 잠금
- 리스크 라벨: 👍 안전 / 🧊 밀당 / 🔥 고위험

## ⑤ 잠긴 카드 클릭
- 결제 X → 공유 유도
- "공유하고 1회 더 받기"

## ⑥ 공유 플로우
- "이 판정… 친구한테 보여주면 무조건 반응 옴ㅋㅋ"

## ⑦ 공유 보상
- 🎁 +1회 즉시 지급
- "하루 최대 2회까지"

## ⑧ 로그인 락인
- 3번째 사용 또는 히스토리 클릭 시
- "지난 판정 다시 보고 싶음?"

## ⑨ 유료 (조건부)
- 조건: 밤10시+ AND 고위험 AND 무료 소진
- "사고 방지용 대체 멘트 3개"
- ₩1,100
